Robot car with PID
November 2018 - Febuari 2022
As a final research project for high-school, did I create a robot with an Arduino, gyroscope and accelerometer. The end result was a robot that drove in a straight line, and corrected itself when it slipped or otherwise just drifted off course. It worked by feeding the measurements from the sensors into a PID which calculated adjustments in motor speed to get the robot back in the correct course

The Loop Programming Language
September 2021 - Ongoing
Creating a Just-In-Time compiled programming language called Loop. Syntax and semantics is similar to something like GoLang and Python. The Language is implemented in Rust and the compilation target currently is LuaJIT.
During the development of the project we worked with a lot of people with different specialisations: media design, infrastructure, software engineering, to name a few.
